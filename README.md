# UtilityBubble

## Installation

- Install Qt5.15.6
```sudo apt-get install qtbase5-dev qtchooser qt5-qmake qtbase5-dev-tools```

- Build Project
```qmake UtilityBubble.pro```
```make```

- On Ubuntu OS
Run as non-root user ```./UtilityBubble```
Run as root user ```sudo ./UtilityBubble``` (MUST FOR TOGGLING CAMERA)
